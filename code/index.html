Pair programming

Pair programming is when two people are working toghether on the same computer to write code. There are two roles. One person is the Driver and one is the Navigator. The Driver is the one writing the code and the Navigator is guiding and keeping the big picture. This method has many great advantages such as learning, mentoring and deveolping. It is important to swtich roles often.
